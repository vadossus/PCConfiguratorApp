# PC Configurator

Веб-сайт, разработанный на HTML, CSS, JavaScript, и PHP с использованием СУБД MySQL.

Представляет собой автоматизированную система подбора комплектующих персонального компьютера.

![alt text](https://github.com/vadossus/pictures/blob/main/Снимок%20экрана%202026-01-19%20204948.png "Изображение 1")

На сайте присутствует конфигуратор ПК, работает он по принципу выбора конфигурации совместимых компонентов компьютера.
![alt text](https://github.com/vadossus/pictures/blob/main/image.png "Изображение 2")


# Для запуска сайта, требуется:
* XAMPP Control Panel

# Чтобы запустить приложение:
Необходимо скинуть файлы в папку xampp\htdocs

(корневой путь будет таким: C:\xampp\htdocs\PCConfiguratorApp)

После включения Apache и MySQL в приложении XAMPP, необходимо запустить сайт через localhost\PCConfiguratorApp
