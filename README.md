* PC Configurator — это веб-приложение для автоматизированного подбора совместимых комплектующих персонального компьютера. Система позволяет пользователям собирать компьютерные конфигурации, гарантируя совместимость выбранных компонентов на основе заложенных в базу данных правил и характеристик.
* Проект реализован с использованием стекa технологий HTML, CSS, JavaScript, PHP и СУБД MySQL.

![alt text](https://github.com/vadossus/pictures/blob/main/Снимок%20экрана%202026-01-19%20204948.png "Изображение 1")

На сайте присутствует конфигуратор ПК, работает он по принципу выбора конфигурации совместимых компонентов компьютера.
![alt text](https://github.com/vadossus/pictures/blob/main/image.png "Изображение 2")

# Технологический стек
* Фронтенд: HTML5, CSS3, Vanilla JavaScript
* Бэкенд: PHP
* База данных: MySQL
* Серверное ПО: Apache (поставляется в составе XAMPP или OsPanel)

# Описание работы конфигуратора
* Пользователь начинает сборку, выбирая один из ключевых компонентов (например, процессор или материнскую плату).
* Система обращается к базе данных, проверяет правила совместимости и обновляет списки доступных для выбора остальных компонентов.
* Пользователь последовательно выбирает все необходимые части ПК.
* На каждом этапе суммируется стоимость конфигурации.
* Итоговая сборка может быть сохранена в системе Избранное пользователя.

# Ключевые особенности
* Конфигуратор динамически фильтрует доступные комплектующие (процессоры, материнские платы, видеокарты, оперативная память, накопители, блоки питания и т.д.) на основе уже выбранных пользователем элементов, исключая несовместимые варианты.

* Интуитивно понятный пошаговый интерфейс с визуальным представлением сборки.

* Централизованное хранилище актуальных компонентов с детальными характеристиками (сокет, форм-фактор, тип памяти, потребляемая мощность и др.).

* Автоматический подсчет общей суммы выбранной конфигурации в реальном времени.

# Для запуска сайта, требуется ПО:
* XAMPP Control Panel или OS Panel

# Чтобы запустить приложение:
## Если это XAMPP:
* Необходимо скинуть файлы в папку xampp\htdocs (корневой путь будет таким: C:\xampp\htdocs\\{название_проекта})
* После включения Apache и MySQL (нажатием на кнопку Start) в приложении XAMPP, необходимо запустить сайт через localhost\\{название_проекта}

## Если это OSPanel:
* Необходимо запустить скинуть проект в папку OsPanel/home/{название_проекта.local}
* Запустить модули MySQL-8.0, PHP 8.2, Apache
* Запустить phpMyAdmin и создать базу данных pc_configurator
* Импортировать файл pc_configurator.sql в одноименную БД
